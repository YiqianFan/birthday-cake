# Birthday Cake - Night Sky Wish

An interactive birthday cake webpage. Blow out the candles by dragging, and watch wishes float across the night sky.

## Features

- Drag to blow out candles (touch + mouse)
- Crescent moon, drifting clouds, fireflies
- Canvas particle system: sparks, shooting stars, fireworks, confetti
- Floating wish text after blowing out candles
- Relight and blow again

## Usage

Open `index.html` in any modern browser. No build step required.
